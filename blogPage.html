1. How do you select an element by its ID in the DOM?

Answer: Use document.getElementById("id") to select an element by its ID.

2. What is the DOM?

Answer: The DOM (Document Object Model) is a tree-like structure representing a webpage, allowing JavaScript to manipulate HTML and CSS.

3. How can you change the content of an HTML element using the DOM?

Answer: You can use element.textContent = "new content"; to change the text inside an element.

4. What is event delegation?

Answer: Event delegation is attaching a single event listener to a parent element to manage events for its child elements.

5. How do you add a class to an element in the DOM?

Answer: Use element.classList.add("className"); to add a class to an element.
